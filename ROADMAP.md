# ROADMAP - A Big Boy's Game

## Phase 1 / V1 (Current Focus)
- Finalize the Enterprise Admin Dashboard (Security, CRM, Margin Scheme, RMA).
- Integrate Live Payment Gateway (Mollie).
- Optimize post-purchase flow (Success Pages).
- Container Deployments on Coolify (Live Environment).

## Phase 2 / V2 (Future Iterations)
- **Logistics Integration**: Implement automated shipping labels and live rates with `Sendcloud` or `MyParcel`.
- **Content Management**: Introduce a generic Text CMS to allow dynamic updating of the About Us, FAQ, and Shop text banners without requiring a developer.
- **Advanced Marketing**: Automated Abandoned Cart recovery emails via Resend.
- **Customer CRM**: Dedicated Customer Accounts, wishlists, and loyalty points system.
